🎬 Netflix India Landing Page Clone
A responsive front-end clone of the Netflix India landing page built using HTML5 and CSS3. This project recreates the look and feel of the official Netflix homepage while focusing on responsive design, layout accuracy, and modern CSS techniques.

⚠️ Disclaimer: This project is created solely for educational and portfolio purposes. It is not affiliated with or endorsed by Netflix.

✨ Features
🎥 Modern Hero Section
📺 Trending Movies Carousel
💜 "More Reasons to Watch" Section
❓ Frequently Asked Questions (FAQ)
📱 Fully Responsive Design
🖱️ Smooth Hover Effects
🦶 Footer with Useful Links
🎨 Clean & Modern User Interface
🛠️ Technologies Used
HTML5
CSS3
Flexbox
CSS Grid
Media Queries
Google Fonts
